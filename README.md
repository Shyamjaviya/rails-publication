## Requirements
- Ruby 3.2
- Rails 8.x (or the version you used)
- Sqlite Database
- Js Assets

## Setup

**Clone the repository**
git clone https://github.com/your-username/your-repo.git
cd your-repo

**Install dependencies**
bundle install

**Setup database**
rails db:create
rails db:migrate

**Start rails server**
## The project will initially start on the port 3000
rails server
